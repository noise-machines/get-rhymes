> Pull great rhymes from RhymeZone and B-Rhymes. Includes near rhymes.

## Install

```console
$ npm install get-rhymes
```

## Usage

```js
const getRhymes = require('get-rhymes')
getRhymes('easy').then(rhymes => {
  console.log(rhymes)
})
// => [..., 'levy', 'busy', 'lazy', ...]
```
